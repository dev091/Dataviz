# Launch Scope

This file captures the current launch-version scope derived from the master roadmap. The governing contract remains [docs/MASTER_ROADMAP.md](docs/MASTER_ROADMAP.md).

## Included In Launch Scope
- Multi-tenant auth, workspaces, RBAC, auditability
- Launch connectors: PostgreSQL, MySQL, structured file uploads, Google Sheets, Salesforce
- Metadata discovery, schema preview, sync runs, retry metadata, scheduled sync hooks
- Dataset catalog with quality profiling and AI Data Prep Autopilot review and governed apply or rollback flow
- Governed semantic KPI layer with metrics, dimensions, joins, calculated fields, ownership, certification controls, synonyms, hierarchies, and versioning
- Autonomous dashboard composition, report packs, executive summaries, KPI scorecards, and saved widgets
- Natural-language analytics grounded in the semantic layer with plan, trace, chart recommendation, and summary
- Proactive intelligence: threshold alerts, anomaly alerts, pacing alerts, trend-break alerts, freshness alerts, digests, suggested actions, audience routing, and investigation paths
- Admin governance, usage metrics, audit logs, AI trust history, and billing scaffolding

## Deferred Beyond Current Launch Scope
- Massive long-tail connector coverage
- Lakehouse, cloud-storage, webhook, and custom connector SDK programs
- Full direct workbook import from incumbent BI tools
- Automated trust comparison against incumbent outputs at migration scale
- Complete metric and transformation lineage across every artifact
- Full prompt and action history across every AI-generated path
- Enterprise SSO/SAML, approval workflows, and collaboration expansion

## Launch Principle
The launch version optimizes for premium wedge strength, fast time-to-value, trust, repeatability, and margin discipline rather than horizontal BI parity.
